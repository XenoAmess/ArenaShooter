Spawn A Weapon

DTD:

<!ELEMENT spawn (vecteur,(usable | gun | melee |shotgun| usabletimer)*,entities?)>
<!ATTLIST spawn spawnperso (false|true) "true"
		        cooldown CDATA #REQUIRED>

Spawn a gun :

<!ELEMENT gun (vecteur, vecteur , entities?)>
<!ATTLIST gun	
				 name CDATA #REQUIRED
				 weight CDATA #REQUIRED
				 pathSprite CDATA #REQUIRED
				 soundPickup CDATA #REQUIRED
				 cooldown CDATA #REQUIRED
				 uses CDATA #REQUIRED           //number of uses/ammo for the gun
				 animPath CDATA #REQUIRED	//null
				 warmupDuration CDATA #REQUIRED
				 soundWarmup CDATA #REQUIRED
				 bangSound CDATA #REQUIRED
				 noAmmoSound CDATA #REQUIRED
				 bulletType CDATA #REQUIRED
				 bulletSpeed CDATA #REQUIRED
				 damage CDATA #REQUIRED
				 cannonLength CDATA #REQUIRED
				 recoil CDATA #REQUIRED
				 thrust CDATA #REQUIRED
				 size CDATA #REQUIRED
				 proba CDATA #REQUIRED> //proba of spawn




take the XML of a map :

Example :

Into a spawn
		<spawn spawnperso="false" cooldown="2">
			<vecteur x="-7" y="1" />
			
		->	<gun name="Minigun" weight="43"
				pathSprite="data/weapons/Minigun_1.png" soundPickup="GunCock_02"
				cooldown="0.03" uses="350" animPath="" warmupDuration="0.25" //time before you can shoot
				soundWarmup="warmup_minigun_01"
				bangSound="Bang1"
				noAmmoSound="no_ammo_01" bulletType="0"
				bulletSpeed="35" damage="1" cannonLength="1.9" recoil="0.2"
				thrust="2.6" size="64" proba="1">

				<vecteur use="handPosL" x="-.8" y="-.1" /> // position of hand where you are taking it
				<vecteur use="handPosR" x="0" y="-.28" />
			</gun> <-

		</spawn>

Congrulation you have create add a gun to this spawner, a 'minigun' with a weight of 43, this sprite : Minigun_1.png and you add a lot of sound for this new gun 
( /!\ for the sound it is not the path just the name without the extention .ogg)

Create a Shotgun : The same but not the same effect

		<shotgun name="shotgun" weight="100"
				pathSprite="data/weapons/shotgun1.png" soundPickup="GunCock_02"
				cooldown="1.2" uses="8" animPath="" warmupDuration="0"
				soundWarmup=""// "" if you don't want to have one
				bangSound="bang_shotgun_01"
				noAmmoSound="no_ammo_01" multiShot="25" dispersion="0.18" bulletType="0"
				bulletSpeed="45" damage="2" cannonLength=".6" recoil="2"
				thrust="25.00" size="2"  proba="1">

				<vecteur use="handPosL" x="-.65" y=".15" />
				<vecteur use="handPosR" x=".067" y=".125" />
			</shotgun>
